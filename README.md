## How to use it
**Requires `Tadpole Operator` role.**

Command | Description
----- | ----- 
`tadpole host` | Host a 2-way connection 
`tadpole join [Key]` | Join one
`tadpole end` | Kill your connection
`tadpole cinfo` | Check your connection
